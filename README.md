
# Sample Telegram Bot
## Version 1.0.2
Telegram bot sample based on https://github.com/tucnak/telebot API

## Installation

Download the library with the standard `go get` command then run `go build` 

```bash
    go github.dev/VinceWhite19/kbot/cmd
```

* `./kbot version` – show bot version
* `./kbot start` – to run the app

    
## Bot Address

https://t.me/vincewhite_bot


## Usage/Examples

* `/start` – show basic bot info
* `/my_id` – get current user id
* `/my_username` – get current username
* `hello` – just a greeting to be polite
